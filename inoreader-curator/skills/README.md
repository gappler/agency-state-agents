# Skills

This folder is empty intentionally. Skills get added when a process has repeated enough that the SOP is real — not preemptively.

Likely first candidates as the agent gets used:

- A "draft react copy" skill that runs the brand voice self-check inline
- A "theme clustering" skill documenting how themes get derived from a batch
- A "feed health" skill for tracking which feeds reliably return content vs. error out

When you create a skill, drop it in this folder as `[skill-name].md`. Reference it from `AGENT.md` if it should be invoked routinely; leave it unreferenced if it's an on-demand tool.
